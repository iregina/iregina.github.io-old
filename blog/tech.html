What is Rails?

Rails is a framework written in the Ruby language that makes programming web applications easier. It is an "opionated" software meaning that it is designed as if there is a best way to program. Some of these concepts are "DRY" method and "Convention over Configuration" Rails is open-sourced under the MIT liscense. Rails was invented by a guy named David Hansson and first realeased July 2004. There is a core team that makes major decisions about how rails will work and overlooks the changes are that being suggested to rails.
