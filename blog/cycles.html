enumerables#cycles

Tell me more about the cycle ruby enumerable methods?

Cycles causes the block of the enumerable element to repeat. You can specify a specific number of times for the block to repeat, or it can go on forever. It will return nil if it goes on forever.

Here's an example:

thing = ["a", "b", "c"]

thing.cycle ( [x] puts x) # print a, b, c, a, b, c, ... (forever)

thing.cycle(3) ( [x] puts x) # print a, b, c, a, b, c, a, b, c
