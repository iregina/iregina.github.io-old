Ruby Classes

What's up with Ruby Classes?

This week we learned about Ruby classes, and it's super fascinating. Classes are an interesting tool to use when you want to create custom methods or modify exisiting methods.

The best analogy would be a purse class. There are many types of purses out in the world, but not all purses are the same. They have different colors, sizes, and textures. However, they all have the function of holding items and being carried by the arm or hands.

In Ruby, classes help organize functions and make it easy to apply the same methods across different objects.
