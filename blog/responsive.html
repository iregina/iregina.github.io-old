responsive

Today, I'm working on building my first responsive web site. As most of my fellow bootcampers tell me, one of the major issues with my current site is that it blows up the moment they descrease the screen size.

Responsive web design uses CSS and html and make sure that the contents of my website looks great on any screen size.

Fixed websites have a set width regardless of the browser size.

Fluid websites are built using percentages for width, and columns are set to be relative to one another.

Adapative websites can target specific device sizes like monitors, tablets, or phones.

Responsive websites have a fluid grid and can use media queries to control the design.
